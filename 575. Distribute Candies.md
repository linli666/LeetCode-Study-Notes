## 575. Distribute Candies
####
**Method 1**
```
class Solution:
    def distributeCandies(self, candies):
        """
        :type candies: List[int]
        :rtype: int
        """
        candies_dict = {}
        half_num = int(len(candies)/2)
        for item in candies:
            if item not in candies_dict:
                candies_dict[item] = 1
            else:
                candies_dict[item] += 1
        num = len(candies_dict)
        if num < half_num:
            return num
```

**Method 2**
```
class Solution:
    def distributeCandies(self, candies):
        """
        :type candies: List[int]
        :rtype: int
        """
        return min(len(candies)//2, len(set(candies)))
```
