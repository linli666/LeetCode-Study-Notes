# 326. Power of Three
####
```
class Solution:
    def isPowerOfThree(self, n):
        """
        :type n: int
        :rtype: bool
        """  
        return n > 0 and (3**19) % n == 0
```
Because the maximum int that computer can input is 2147483648. And 3**20 will exeed this value.  
So 3**19 will be the maximum value that can input.
