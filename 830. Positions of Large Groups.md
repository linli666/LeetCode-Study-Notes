# 830. Positions of Large Groups
####
```
class Solution:
    def largeGroupPositions(self, S):
        """
        :type S: str
        :rtype: List[List[int]]
        """
				last_s = ''
        last_ix = 0
        out_put = []
        for ix, item in enumerate(S):
            if item == last_s:
                continue
            else:
                last_s = item
                if (ix - last_ix) >= 3:
                    out_put.append([last_ix,ix-1])
                last_ix = ix
        if item == last_s:
            if ix - last_ix+1 >= 3:
                out_put.append([last_ix,ix])
        return out_put
```
